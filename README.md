# 🧑‍💼 Employee Management System (MERN Stack)

A full-stack Employee Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This application helps manage employee data with authentication and role-based access.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 👨‍💼 Employee Management (Add, Edit, Delete, View)
- 🔎 Search & Filter Employees
- 🛡️ Role-Based Access (Admin/User)
- 📊 Dashboard Overview
- 🌐 REST API Integration

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Axios
- Bootstrap / Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
